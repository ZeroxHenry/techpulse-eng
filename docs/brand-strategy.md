## 1. 브랜드 정체성

### 이름

| 항목 | 내용 |
|------|------|
| **채널명** | 0xH |
| **발음** | 헥세이치 |
| **의미** | 0x(hex, 엔지니어의 언어) + H(Henry) |
| **블로그** | StackPulse (0xhenry.dev) |

### 한 줄 정의

> **"로봇 엔지니어가 직접 부딪히며 배운 것을, 강의처럼 정리해서 공유하는 채널"**

### 핵심 가치

1. **실전 우선** — 이론만이 아니라, 실제로 해봤을 때 생기는 문제와 해결
2. **정직한 과정** — 삽질, 실패, 돌아간 길도 숨기지 않음
3. **구조화된 전달** — 강의형 PPT로 "문제 → 개선 → 결과"를 명확하게
4. **엔지니어 to 엔지니어** — 초보한테 쉽게가 아니라, 같은 눈높이에서 나누기

### 차별점

```
기존 기술 유튜버:          0xH:
이론 설명 위주         →   실제 프로젝트에서 부딪힌 문제 기반
"따라하세요"           →   "이런 문제가 있었고, 이렇게 해결했습니다"
단일 분야              →   로봇공학 중심 + HW/SW/AI 풀스택
얼굴 or 화면만         →   VTuber 아바타 + PPT 강의형
```

## 2. 타겟 시청자

### 주요 타겟

| 페르소나 | 설명 | 니즈 |
|----------|------|------|
| **주니어 로봇 엔지니어** | 회사/연구실에서 로봇 개발 시작한 1-3년차 | 실전 트러블슈팅, 장비 선정 가이드 |
| **임베디드 입문자** | STM32, Jetson 등 시작하려는 개발자 | 어디서부터 시작해야 하는지, 실제 세팅 과정 |
| **풀스택 엔지니어** | HW/SW 둘 다 관심 있는 메이커 | 로봇 + 웹 + AI 통합 프로젝트 |
| **대학생/대학원생** | 로봇공학, 전자공학 전공 | 수업에서 안 알려주는 실전 |

### 언어 전략

```
YouTube (0xH)         →  한국어 메인 + 영어 자막
StackPulse 블로그      →  한국어 + 영어 (둘 다 작성)
```

**한국어 메인인 이유:**
- 한국어 로봇공학/임베디드 강의형 콘텐츠 = 블루오션
- 영어 로봇공학 유튜브는 이미 포화 (레드오션)
- 한국어로 먼저 팬층 확보 → 영어 자막으로 글로벌 노출
- 블로그(StackPulse)가 영어 커버하니 역할 분담 가능

**향후 확장 시나리오:**
1. 초반: 한국어 영상 + 영어 자막
2. 중기: 영어 자막 퀄리티 높이기 (직접 검수)
3. 장기: 영어 전용 채널 or 영어 더빙 버전 (구독자 규모 보고 판단)

## 3. 콘텐츠 전략

### 콘텐츠 4대 축 (Pillar)

| 축 | 비중 | 설명 | 예시 |
|----|------|------|------|
| **로봇공학/임베디드** | 40% | 메인 정체성, 핵심 콘텐츠 | Jetson, ZED, ROS2, STM32, RTOS |
| **하드웨어/PCB** | 20% | 실물 다루는 매력, 영상 차별화 | KiCad, 3D프린팅, 센서보드 설계 |
| **AI/ML** | 20% | 트렌드 + 로봇에 적용 | 컴퓨터비전, depth sensing, LLM 로봇 제어 |
| **소프트웨어/웹** | 20% | StackPulse 자체가 포트폴리오 | Next.js, 개발 인프라, 자동화 |

### 영상 포맷

**메인: 강의형 (주력)**
- 문제 → 개선 → 결과 구조
- PPT + 화면녹화 + VTuber 아바타
- 15-20분
- 주 1회

**서브 포맷 (추후 추가 가능):**

| 포맷 | 길이 | 주기 | 설명 |
|------|------|------|------|
| **Shorts/숏폼** | 30-60초 | 주 1-2회 | 핵심 팁 하나, 에러 해결 한 줄 |
| **프로젝트 시리즈** | 20-30분 | 시리즈당 3-5편 | 하나의 로봇 프로젝트 처음부터 끝까지 |
| **장비 리뷰** | 10-15분 | 월 1회 | 실제로 써본 보드/센서/도구 솔직 리뷰 |
| **월간 회고** | 10분 | 월 1회 | 이번 달 배운 것, 다음 달 계획 |
